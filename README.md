Portafolio Web (Certamen 1)- Simón Mellado Guerra

Este proyecto corresponde a un portafolio web personal, para el certamen 1 de "Programación Front End".
El objetivo pricipal es mostrar la información profesional, habilidades técnicas, proyectos y un medio de contacto en una interfaz.

El sitio está construido utilizando HTML5 y CSS3, aplicando buenas prácticas de diseño web mostradas en clases.

**Tecnologias utilizadas:**

- HTML5
- CSS3
  - Flexbox
  - CSS Grid
- Google Fonts
  - Noto Sans
  - Lato
